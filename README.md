## The short version

I build scripts, systems and UIs for **GTA V (FiveM)** and **Red Dead Redemption 2 (RedM)** roleplay servers. If your server needs an economy that doesn't collapse, a heist that doesn't desync, or a saloon that actually feels alive — that's my corner of the map.

- 🤠 **Right now:** knee-deep in RedM — VORP & RSG framework work
- 🏙️ **Before that:** a whole lot of QB-Core and ESX for FiveM cities
- 🌐 **On the side:** websites and NUI interfaces, because a good script deserves a good face
